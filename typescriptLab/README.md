Typescript set up:
```
    npm install -g typescript
    tsc --version
```

node initialization:
    `npm init`

Type definition:
    `npm install @types/node --save-dev`

Init tsconfig.json:
    `tsc --init`

Transpiling:
```
    tsc #just transpile everything
    tsc filename.ts #transpile filename.ts
    tsc --outDir transpiled/
```
